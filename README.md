# Claude Nonprofit Assistant

## Overview

Claude Nonprofit Assistant is a simple AI-powered application designed to help nonprofit organizations answer common questions related to volunteering, donations, services, and community programs.

## Features

* Answers common nonprofit questions
* Provides volunteer information
* Provides donation information
* Displays organization details
* User-friendly interface built with Streamlit

## Technologies Used

* Python
* Streamlit
* GitHub

## Project Structure

## Sample Questions

* How can I volunteer?
* What are your operating hours?
* What services do you provide?
* How can I donate?

## Purpose

This project demonstrates how AI-inspired assistants can support nonprofit organizations by reducing repetitive administrative tasks and improving access to information.
