# Claude Nonprofit Assistant

## Overview
A Claude-powered AI assistant designed to help nonprofit organizations automate responses to common volunteer, donation, and service-related questions.

## Features
- AI-powered question answering
- Volunteer information support
- Donation information support
- Service and program FAQs

## Technologies
- Python
- Claude API
- Streamlit

## Purpose
This project demonstrates how AI can help nonprofit organizations reduce repetitive administrative work and improve access to information.
